Como Usar os Exercícios

1. Leia o Problema

Cada exercício tem um README.md com descrição completa do problema.

2. Analise a Solução

Cada README inclui um diagrama PlantUML sugerindo a estrutura da solução.

3. Implemente

Crie a implementação em TypeScript seguindo o padrão e os critérios de avaliação.
